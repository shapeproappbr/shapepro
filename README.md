# ShapePro

Plataforma de gestao para personal trainers e studios de academia.

## Funcionalidades
- Gestao de alunos e fichas de treino
- Controle de pagamentos e mensalidades
- Agenda de sessoes e aulas
- Acompanhamento de evolucao fisica
- App para o aluno acompanhar treinos

## Stack
PHP 8.2 | MySQL | Apache2 | Debian 12

## Configuracao do ambiente

### Requisitos
- PHP 8.2+
- MySQL 8+
- Composer
- Git

### Instalacao
```bash
git clone git@github.com:SEU_USUARIO/shapepro.git
cd shapepro
cp .env.example .env
# edite .env com suas configuracoes
```

## Contribuindo
1. Crie uma branch: `git checkout -b feature/minha-feature`
2. Faca commits descritivos: `git commit -m "feat: adiciona tela de alunos"`
3. Abra um Pull Request para `develop`

Ver `CLAUDE.md` para contexto completo do projeto (arquitetura, modulos, convencoes).
